# 2026-04-01 Session

## Decisions

### All-layer tokenization-aware Qwen geometry probe
**What:** Implemented a next-generation Qwen anchor geometry probe that runs across all 28 model layers, audits tokenization at the anchor span, and evaluates both full-span and trimmed-span geometry metrics without modifying tokenizer or steering logic.
**Why:** The goal was to test whether content-like vs procedure-like anchor polarity survives explicit tokenization controls and first-token trimming, using hidden-state delta geometry only.
**Impact:** Repo now has an all-layer analysis script, updated geometry utilities, stronger anchor-geometry tests, and a new report showing that full-span gives only partial separation while trimmed-span collapses to no reliable separation because only 2/12 cases retain full geometry after trimming.

<details>
Context
A prior 4-layer geometry probe suggested partial separation between content-like and procedure-like anchors on Qwen/Qwen2.5-1.5B, but it did not control explicitly for tokenization artifacts or first-token effects.

Options considered
1. Change the prompt set aggressively to equalize token counts.
2. Keep the existing prompt set and add strict tokenization audit plus trimmed-span control.
3. Redesign tokenizer/model instrumentation.

Decision
Chose option 2. Kept the existing paired prompt set because all anchor groups already tokenize cleanly and consistently within group. Added an all-layer probe over model layers 0..27, recorded tokenization audit fields per case, and computed geometry metrics in full-span and trimmed-span modes. Used trimmed-span as the stricter token-boundary control without changing the tokenizer.

Tradeoffs
- This preserves comparability with the earlier probe and avoids architectural changes.
- However, trimming short 4-token anchors leaves only 3 tokens, so coherence/tortuosity/rank-1 EV become unavailable for 10/12 cases. That makes trimmed-span a strong control but also a harsh information bottleneck.

Follow-up
The best next experiment is a tightly paired paraphrase probe with anchors that remain >=5 tokens after trimming, so both content-like and procedure-like groups retain full geometry metrics under the same control.
</details>

## Learnings

### Long-anchor rerun removed trim-collapse confound
**What:** Reran the all-layer Qwen anchor geometry probe with longer 5-6 token anchors so every case retained full geometry metrics after trimmed-span control; the polarity signal still did not survive tokenization controls.
**Why:** The prior probe was inconclusive because trimming collapsed metric availability for 10/12 cases. The long-anchor rerun isolated whether the negative result was just a short-span artifact.
**Impact:** Current best evidence is stronger against the geometry-only polarity hypothesis in this form: tokenization audit stayed fully clean, all 12 cases preserved trimmed geometry, yet trimmed-span still showed no separation and full-span only weak partial separation.

<details>
Context
The first tokenization-aware all-layer probe used 4-5 token anchors. After trimming the first token, only 2/12 cases retained enough tokens for coherence/tortuosity/rank-1 metrics, so trimmed-span evidence was too sparse.

Decision
Updated the prompt set to use longer anchors:
- strictly vegan meal plan policy
- async FastAPI service architecture policy
- JSON only response format policy
- proof by contradiction reasoning steps
- binary search update loop procedure
- dependency injection request flow sequence
Then reran the same probe without changing tokenizer, model, or geometry code.

Results
- Tokenization audit: 12 clean, 0 noisy, 0 skipped.
- Cases retaining full geometry after trimming: 12/12.
- Full-span verdict remained only partial separation.
- Trimmed-span verdict remained no separation.
- In trimmed-span mode: coherence positive on 13/28 layers, tortuosity positive on 0/28, rank-1 EV positive on 1/28, cross-prompt stability positive on 1/28.

Interpretation
The negative trimmed-span result is no longer explainable by short-span metric collapse. That substantially weakens the polarity-from-geometry hypothesis in its current form.
</details>
